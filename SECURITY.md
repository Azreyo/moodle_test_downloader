# Security Policy

## Supported Versions

The following table lists the versions of this repository that are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 0.1   | :x:                |

## Reporting a Vulnerability

To report a security vulnerability in this repository, please open an issue and label it as "security".  
If you prefer private disclosure, email the maintainers directly.

You will receive an initial response within 7 days.  
If the vulnerability is accepted, we will coordinate a fix and notify you when it is resolved.  
If the vulnerability is declined, we will provide an explanation.

Thank you for helping keep this repository secure!
